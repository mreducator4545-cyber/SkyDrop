# Skydrop 🚀

A full-stack file-sharing application built with the MERN stack.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Storage:** Local Server Uploads (can be updated to AWS S3/Cloudinary)

## 📁 Project Structure
- `/client`: Frontend React application.
- `/server`: Backend Node.js API and file storage.

## 🚀 Getting Started

### Prerequisites
- Node.js installed on your machine.
- A terminal (Command Prompt, Terminal, or VS Code terminal).

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com
   cd skydrop
   ```

2. **Setup the Backend**
   ```bash
   cd server
   npm install
   npm start
   ```

3. **Setup the Frontend**
   In a new terminal window:
   ```bash
   cd client
   npm install
   npm run dev
   ```

## 📝 Features
- Multi-file upload support.
- Modern UI styled with Tailwind CSS.
- Automated file naming with timestamps.
